# 柯林斯词汇学习器 (Collins Vocabulary Learner)

基于 Streamlit 的便携版词汇学习工具，支持本地全功能运行与 Streamlit Cloud 云端部署。

## 功能特点

- 格林斯分级词汇（Level 1~5）学习
- 柯林斯词频筛选（五星~二星）
- 主词 + 三个同义词：音标、翻译、真人发音（有道/缓存/GitHub CDN）
- 纯听模式：自动循环播放
- 📝 富文本笔记（本地 + 云端 GitHub 同步）
- 🎙️ 录音（仅本地）
- 🧠 AI 练习题（OpenAI 生成 5 道题）
- 📦 全局/增量备份与恢复
- 📥 手机端 MP3 下载
